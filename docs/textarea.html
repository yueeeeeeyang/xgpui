Textarea 多行输入组件

Textarea 是标准多行文本输入组件，支持换行、软换行、IME、选区、剪贴板、 鼠标定位、拖拽选中、内部滚动条、最大长度、只读、禁用、状态样式和 helper text。

Multiline IME Unicode Grapheme Rows / Max Rows Light / Dark

基础用法

Textarea 是 gpui entity 组件，通常在父视图构造函数中通过 cx.new 创建，再在 Render 中作为子元素渲染。 

/// 示例父视图，持有一个 Textarea 实体。
struct FeedbackView {
    comment: Entity<Textarea>,
}

impl FeedbackView {
    /// 创建多行输入框并配置占位文本、最大长度、行数和变化回调。
    fn new(cx: &mut Context<Self>) -> Self {
        let comment = cx.new(|cx| {
            Textarea::new(
                cx,
                TextareaProps::default()
                    .placeholder("请输入反馈内容")
                    .rows(4)
                    .max_rows(Some(8))
                    .max_length(Some(500))
                    .helper_text(Some(SharedString::from("支持换行、中文输入法和内部滚动")))
                    .on_change(|value| {
                        // 在这里同步表单状态或触发上层业务逻辑。
                        println!("当前内容：{value}");
                    }),
            )
        });

        Self { comment }
    }
}

TextareaProps

TextareaProps::default() 提供默认配置，可通过链式方法覆盖需要的属性。

属性 类型 说明
value SharedString 初始多行文本。创建后需要外部同步时使用 set_value。
placeholder SharedString 空值时显示在首行位置的占位文本。
disabled bool 禁用后不能聚焦、选择、复制或编辑。
readonly bool 只读后可以聚焦、选择和复制，但不能修改、粘贴、剪切或清除。
required bool 显示必填语义标记。当前版本不执行内置校验。
max_length Option<usize> 最大长度按 Unicode 字素簇计算，换行、emoji 和组合字符都计入长度。
rows / min_rows / max_rows usize / Option<usize> 控制可见行数；超过最大行数后在组件内部滚动。
size TextareaSize Small、Medium、Large，影响字号、行高和内边距。
variant TextareaVariant Outlined、Filled、Ghost，影响背景和边框展示。
status TextareaStatus Default、Error、Warning、Success，只负责语义颜色。
helper_text Option<SharedString> 输入框下方辅助文本，颜色会跟随语义状态。
on_submit FnMut(SharedString) Cmd+Enter / Ctrl+Enter 触发提交，普通 Enter 仍插入换行。

公开方法

方法 用途
Textarea::new(cx, props) 创建组件状态。
value() 读取当前文本值。
set_value(value, cx) 从外部同步文本值，不触发 on_change，避免同步回环。
set_disabled(disabled, cx) 同步禁用状态；禁用会停止光标闪烁和拖选自动滚动。
set_readonly(readonly, cx) 同步只读状态，不改变当前文本和选区。
set_status(status, cx) 同步语义状态，只影响视觉样式。
set_helper_text(helper_text, cx) 同步辅助文本，可设置或清空。
clear(cx) 清空文本，并在内容变化时触发 on_change。
select_all(cx) 选中全部文本。
focus_handle() 返回 gpui 焦点句柄，便于父组件主动聚焦或协调焦点。

键盘与输入行为

文本编辑

  • Enter 和 Shift+Enter 插入 \n。
  • Cmd+Enter / Ctrl+Enter 触发 on_submit，不插入换行。
  • 支持 Backspace、Delete、方向键、Home、End、全选、复制、剪切和粘贴。

选择与滚动

  • 支持鼠标定位、拖选和 Shift 扩展选择。
  • 内容超过最大可见行数后在组件内部滚动。
  • 光标移动时会自动调整滚动位置，保持光标可见。

Unicode 处理

  • 光标移动和平台替换范围按 Unicode 字素簇边界规范化。
  • max_length 按用户可感知字素簇计数。
  • 内部包含 UTF-8 和 UTF-16 区间转换，用于平台输入协议。

状态限制

  • disabled 禁止聚焦、选择、复制和编辑。
  • readonly 允许聚焦、选择和复制，但禁止修改。
  • 第一版不支持右下角拖拽 resize、密码/数字类型或前后缀 slot。
