Select 单选下拉组件

Select 是单选下拉组件，支持本地搜索、锚定下拉面板、键盘导航、清除、禁用、状态样式、 helper text、外部同步和明暗皮肤。第一版不包含多选、远程加载、分组选项或自定义 option 渲染。

Select 基础用法

Select 是 gpui entity 组件，通常在父视图中通过 cx.new 创建。第一版只支持单选、本地搜索和文本 option，不支持多选、远程加载、分组或自定义 option 渲染。 

/// 示例父视图，持有一个 Select 实体。
struct FormView {
    city: Entity<Select>,
}

impl FormView {
    /// 创建城市选择框并配置选项、搜索、清除和变化回调。
    fn new(cx: &mut Context<Self>) -> Self {
        let city = cx.new(|cx| {
            Select::new(
                cx,
                SelectProps::default()
                    .placeholder("请选择城市")
                    .options(vec![
                        SelectOption::new("beijing", "北京"),
                        SelectOption::new("shanghai", "上海"),
                        SelectOption::new("disabled", "不可选城市").disabled(true),
                    ])
                    .searchable(true)
                    .clearable(true)
                    .helper_text(Some(SharedString::from("支持键盘导航和本地搜索")))
                    .on_change(|value| {
                        // 在这里同步表单状态或触发上层业务逻辑。
                        println!("当前选择值：{value:?}");
                    }),
            )
        });

        Self { city }
    }
}

impl Render for FormView {
    /// 渲染父视图内容。
    fn render(&mut self, _window: &mut Window, _cx: &mut Context<Self>) -> impl IntoElement {
        div().child(self.city.clone())
    }
}

SelectOption::value 建议在同组选项中保持唯一。如果出现重复值，组件会按第一个 匹配项展示和选择，避免运行时额外报错破坏现有数据流。

SelectProps

SelectProps::default() 提供默认配置，可通过链式方法覆盖需要的属性。

属性 类型 说明
value Option<SharedString> 初始选中值。创建后需要外部同步时使用 set_value。
placeholder SharedString 未选择时显示的占位文本。
options Vec<SelectOption> 下拉选项列表。SelectOption 包含 value、label 和 disabled。
disabled bool 禁用后不能聚焦、打开、选择或清除。
clearable bool 有值且非禁用时显示清除按钮。
required bool 显示必填语义标记。当前版本不执行内置校验。
searchable bool 打开面板后直接在原选择框内输入内容，按 label 做大小写不敏感子串匹配。
search_placeholder SharedString 搜索模式下原选择框为空时显示的占位文本。
size SelectSize Small、Medium、Large，影响触发器和选项尺寸。
variant SelectVariant Outlined、Filled、Ghost，影响触发器背景和边框。
status SelectStatus Default、Error、Warning、Success，只负责语义颜色。
helper_text Option<SharedString> 选择框下方辅助文本，颜色会跟随语义状态。
max_popup_height Pixels 下拉面板最大高度，超过后内部滚动。
empty_text SharedString 选项为空或搜索结果为空时展示的文本。
on_change FnMut(Option<SharedString>) 用户选择或清除导致值变化时触发，外部 set_value 不触发。
on_open_change FnMut(bool) 下拉面板打开状态变化时触发。
on_search_change FnMut(SharedString) 搜索词变化时触发。

Select 公开方法

方法 用途
Select::new(cx, props) 创建组件状态。
value() 读取当前选中值。
set_value(value, cx) 从外部同步选中值，不触发 on_change，避免同步回环。
clear(cx) 清空当前选择，并在值变化时触发 on_change。
open(cx) / close(cx) / toggle(cx) 从外部控制下拉面板打开状态。
focus_handle() 返回 gpui 焦点句柄，便于父组件主动聚焦或协调焦点。 
self.city.update(cx, |select, cx| {
    // 外部写入值不会触发 on_change，适合受控表单回填场景。
    select.set_value(Some(SharedString::from("shanghai")), cx);
});

Select 键盘行为

打开与选择

  • Enter / Space：关闭时打开，打开时选择当前高亮项。
  • Escape：关闭下拉面板。
  • Tab：保留系统焦点切换行为，同时关闭下拉面板。

导航

  • ArrowDown / ArrowUp：打开面板并移动高亮项。
  • Home / End：移动到第一个或最后一个可选项。
  • 键盘高亮会跳过禁用选项。

搜索

  • searchable = true 时，打开面板会清空搜索词。
  • 输入普通字符会直接显示在原选择框内，并按 label 进行大小写不敏感本地过滤。
  • 搜索编辑由 Select 内部实现，支持光标闪烁、移动、选区、拖拽选中、复制和粘贴。
  • Backspace 会删除搜索词末尾的一个 Unicode 字素簇。

面板

  • 下拉面板通过 gpui::anchored() 锚定到触发器下方。
  • 超过 max_popup_height 后内部滚动。
  • 点击组件外部会关闭面板；当前版本不主动做上下翻转。
