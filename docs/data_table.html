DataTable 标准数据表格组件

DataTable<T> 提供本地全局过滤、单列排序、内部分页、行选择、 自定义展示/操作列、loading/empty、禁用、状态样式、helper text、键盘导航、 虚拟行渲染和明暗皮肤。

Filter Sort Pagination Select Custom Columns Virtual Rows Light / Dark

基础用法

DataTableProps::new(row_key) 必须传入稳定 row key 生成器。过滤、排序、 分页和选择都会改变显示位置，选择回调和受控同步都以该 key 为准。 

#[derive(Clone)]
struct Order {
    id: &'static str,
    customer: &'static str,
    status: &'static str,
}

let table = cx.new(|cx| {
    DataTable::new(
        cx,
        DataTableProps::new(|row: &Order| SharedString::from(row.id))
            .rows(orders)
            .columns(vec![
                DataTableColumn::text("customer", "客户", |row: &Order| SharedString::from(row.customer)),
                DataTableColumn::text("status", "状态", |row: &Order| SharedString::from(row.status)),
                DataTableColumn::actions("actions", "操作", |ctx| {
                    div().child(if ctx.selected { "已选中" } else { "查看" }).into_any_element()
                }),
            ])
            .selection_mode(DataTableSelectionMode::Multiple)
            .page_size(20),
    )
});

列模型

构造器 用途 过滤 / 排序
DataTableColumn::text(key, title, accessor) 类型化文本数据列，accessor 从行数据读取 SharedString。 默认参与全局过滤和表头排序。
DataTableColumn::display(key, title, renderer) 自定义展示列，renderer 返回 AnyElement。 默认不参与过滤或排序。
DataTableColumn::actions(key, title, renderer) 操作列便捷构造，默认右对齐，适合放在表格最后。 不可过滤、不可排序。

第一版数据处理顺序固定为：rows -> 全局过滤 -> 单列排序 -> 分页 -> 当前页可见行。 自定义展示列只负责渲染，不会影响过滤和排序结果。

DataTableProps

属性 类型 说明
rows / columns Vec<T> / Vec<DataTableColumn<T>> 原始行数据和列配置。重复 row key 会保留第一次出现的行。
row_disabled impl Fn(&T) -> bool 禁用行仍展示，但不会参与行选择、表头全选或键盘选择。
filter_text / show_filter / filter_placeholder SharedString / bool / SharedString 内置全局过滤输入框配置，过滤大小写不敏感。
sort Option<DataTableSort> 单列排序状态；不可排序列会被规范化为 None。
page / page_size / page_size_options usize / usize / Vec<usize> 内部分页状态，底部每页条数使用 Select 展示，过滤、排序或数据变化后会自动夹到合法页码。
selection_mode / selected_row_keys DataTableSelectionMode / Vec<SharedString> None、Single 或 Multiple 行选择。
loading / disabled / required bool 加载、禁用和必填语义。禁用或加载时会阻止过滤、排序、分页和选择交互。
size / variant / status DataTableSize / DataTableVariant / DataTableStatus 控制行高、容器样式和语义状态色。
helper_text / empty_text / loading_text Option<SharedString> / SharedString 辅助文本、空状态文本和加载状态文本。
max_height Pixels 当前页虚拟列表最大高度，超过后内部滚动。

公开方法

rows() / filtered_row_count() / page_state() 读取原始行、过滤后行数和当前分页派生状态。
sort() / filter_text() / selected_row_keys() 读取当前排序、过滤文本和 selected row key。
set_rows(...) / set_columns(...) 从外部同步数据和列配置，不触发交互回调。
set_filter_text(...) / set_sort(...) 同步过滤和排序状态，不触发 on_filter_change 或 on_sort_change。
set_page(...) / set_page_size(...) 同步分页状态，并同步底部每页条数 Select，不触发 on_page_change。
set_selected_row_keys(...) / clear_selection(cx) 同步或按交互语义清空 selected row key。
set_loading(...) / set_disabled(...) / set_status(...) / set_helper_text(...) 同步展示输入。受控同步不会触发用户交互回调。
clear_filter(cx) / focus_handle() 清空过滤文本并返回焦点句柄。

键盘行为

  • Up / Down 在当前页移动活动行。
  • Home / End 移动到当前页首个或最后一个可选行。
  • Enter / Space 执行当前活动行选择。
  • Cmd+A / Ctrl+A 在多选模式下切换当前页可选行全选。

第一版不包含远程数据模式、列拖拽、固定列、列显示控制、列级筛选弹层、树形表格或单元格编辑。
