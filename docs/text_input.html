TextInput 单行输入组件

TextInput 是完整的单行文本输入组件，支持普通文本、密码、数字、IME、选区、剪贴板、 鼠标定位、拖拽选中、自动滚动、清除、只读、禁用、状态样式和前后缀插槽。

Text / Password / Number IME Unicode Grapheme Light / Dark

基础用法

TextInput 是 gpui entity 组件，通常在父视图构造函数中通过 cx.new 创建，再在 Render 中作为子元素渲染。 

/// 示例父视图，持有一个 TextInput 实体。
struct FormView {
    username: Entity<TextInput>,
}

impl FormView {
    /// 创建输入框并配置占位文本、清除按钮、最大长度和变化回调。
    fn new(cx: &mut Context<Self>) -> Self {
        let username = cx.new(|cx| {
            TextInput::new(
                cx,
                TextInputProps::default()
                    .placeholder("请输入用户名")
                    .clearable(true)
                    .max_length(Some(20))
                    .helper_text(Some(SharedString::from("最多输入 20 个字素簇")))
                    .on_change(|value| {
                        // 在这里同步表单状态或触发上层业务逻辑。
                        println!("当前输入值：{value}");
                    }),
            )
        });

        Self { username }
    }
}

impl Render for FormView {
    /// 渲染父视图内容。
    fn render(&mut self, _window: &mut Window, _cx: &mut Context<Self>) -> impl IntoElement {
        div().child(self.username.clone())
    }
}

输入类型

TextInputType 用于配置输入框内容语义。当前支持 Text、 Password 和 Number；组件值始终是 SharedString， 不会因为类型不同而返回掩码文本或数值类型。 

let password = cx.new(|cx| {
    TextInput::new(
        cx,
        TextInputProps::default()
            .input_type(TextInputType::Password)
            .placeholder("请输入密码")
            .clearable(true),
    )
});

let amount = cx.new(|cx| {
    TextInput::new(
        cx,
        TextInputProps::default()
            .input_type(TextInputType::Number)
            .placeholder("请输入金额"),
    )
});

Password

  • 默认按 Unicode 字素簇显示为掩码字符。
  • 非禁用状态下始终显示眼睛按钮，可切换真实内容展示。
  • value、on_change、复制和剪切都使用真实文本。

Number

  • 允许空值、负号、小数点以及 1.、-.5 等输入中间态。
  • 拒绝字母、多小数点和错误位置的负号。
  • 不做 min、max、step、格式化或数值解析。

TextInputProps

TextInputProps::default() 提供默认配置，可通过链式方法覆盖需要的属性。

属性 类型 说明
value SharedString 初始输入值。创建后需要外部同步时使用 set_value。
placeholder SharedString 空值时显示的占位文本。
disabled bool 禁用后不能聚焦、选择或编辑。
readonly bool 只读后可以聚焦、选择和复制，但不能修改、粘贴、剪切或清除。
clearable bool 有值且非禁用、非只读时显示清除按钮。
required bool 显示必填语义标记。当前版本不执行内置校验。
max_length Option<usize> 最大长度按 Unicode 字素簇计算，粘贴和 IME 输入都会遵守该限制。
size TextInputSize Small、Medium、Large，影响高度、字号、行高和内边距。
variant TextInputVariant Outlined、Filled、Ghost，影响背景和边框展示。
status TextInputStatus Default、Error、Warning、Success，只负责语义颜色。
input_type TextInputType Text、Password、Number，控制密码掩码和数字输入限制。
helper_text Option<SharedString> 输入框下方辅助文本，颜色会跟随语义状态。
prefix Option<TextInputSlot> 前缀插槽，每次渲染生成新的 gpui 元素。
suffix Option<TextInputSlot> 后缀插槽，每次渲染生成新的 gpui 元素。
on_change FnMut(SharedString) 内容确实变化时触发，光标移动、选区变化和焦点变化不会触发。
on_focus / on_blur FnMut() 聚焦和失焦回调。
on_enter FnMut(SharedString) 按下回车时传出当前完整文本，适合搜索和简单提交。
on_key_down FnMut(Keystroke) 透出 gpui 按键事件，方便上层记录或响应特殊按键。

前后缀插槽

TextInputSlot 保存的是可重复调用的闭包，每次渲染都会生成新的 AnyElement。这样可以避免持有一次性 gpui element 引发的生命周期和重复渲染问题。 

let url_input = cx.new(|cx| {
    TextInput::new(
        cx,
        TextInputProps::default()
            .placeholder("example.com")
            .prefix(Some(TextInputSlot::new(|| {
                // 前缀可以是任意 gpui 元素。
                div().child("https://").into_any_element()
            })))
            .suffix(Some(TextInputSlot::text(".com")))
            .clearable(true),
    )
});

公开方法

方法 用途
TextInput::new(cx, props) 创建组件状态。
value() 读取当前文本值。
set_value(value, cx) 从外部同步文本值，不触发 on_change，避免同步回环。
clear(cx) 清空文本，并在内容变化时触发 on_change。
focus_handle() 返回 gpui 焦点句柄，便于父组件主动聚焦或协调焦点。
select_all(cx) 选中全部文本。
move_to_end(cx) 把光标移动到文本末尾。 
self.username.update(cx, |input, cx| {
    // 外部写入值后移动到末尾，适合受控表单回填场景。
    input.set_value("由父组件写入", cx);
    input.move_to_end(cx);
});

键盘与输入行为

文本编辑

  • 支持单行输入，粘贴多行文本时会把换行规范化为空格。
  • 支持 Backspace、Delete、方向键、Home、End、全选、复制、剪切和粘贴。
  • 支持中文输入法 marked text 和候选框范围定位。
  • 密码类型只改变显示文本，不改变真实值和剪贴板文本。
  • 数字类型会拒绝不符合数字形态的键盘输入、粘贴和 IME 提交。

选择与滚动

  • 支持鼠标定位、拖选和 Shift 扩展选择。
  • 拖选到输入框边缘时会自动横向滚动并继续扩展选区。
  • 光标静止后会闪烁，移动或输入时会恢复常亮。

Unicode 处理

  • 光标移动按 Unicode 字素簇边界处理。
  • max_length 按用户可感知字素簇计数。
  • 内部包含 UTF-8 和 UTF-16 区间转换，用于平台输入协议。

状态限制

  • disabled 禁止聚焦、选择和编辑。
  • readonly 允许聚焦、选择和复制，但禁止修改。
  • status 只展示语义颜色，不内置表单校验器。
