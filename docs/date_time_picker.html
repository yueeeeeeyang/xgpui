DateTimePicker 日期时间选择器

DateTimePicker 支持日期、时间、日期时间以及对应范围选择。组件使用 chrono naive 类型，不处理时区；手动输入使用严格格式解析，弹层提供日历、 秒级时间列、清空和确认操作。

Date / Time / DateTime Range Strict Parse Naive Chrono Light / Dark

基础用法

DateTimePickerProps 默认使用 Date 模式。范围模式对外只暴露完整范围； 弹层里的半选状态只作为内部草稿存在。 

let picker = cx.new(|cx| {
    DateTimePicker::new(
        cx,
        DateTimePickerProps::default()
            .mode(DateTimePickerMode::DateTimeRange)
            .value(Some(DateTimePickerValue::DateTimeRange(DateTimeRange::new(
                NaiveDateTime::new(start_date, start_time),
                NaiveDateTime::new(end_date, end_time),
            ))))
            .helper_text(Some(SharedString::from("选择业务时间范围"))),
    )
});

模式和值

模式 值类型 提交语义
Date DateTimePickerValue::Date(NaiveDate) 点击日期后立即提交并关闭弹层。
Time DateTimePickerValue::Time(NaiveTime) 选择时间草稿，点击确认后提交。
DateTime DateTimePickerValue::DateTime(NaiveDateTime) 日期和时间共同形成草稿，点击确认后提交。
DateRange / TimeRange / DateTimeRange DateTimeRange<T> 起止值会规范化为 start <= end，点击确认后提交。

DateTimePickerProps

mode / value 选择模式和初始值。模式不匹配的值会被规范化为空。
min / max / disabled_date / disabled_time 提交约束和禁用规则，手动输入与弹层选择共用同一套校验。
date_format / time_format / datetime_format / range_separator 展示和严格手动输入解析使用同一套格式。
minute_step / second_step 时间列步长，默认均为 1，因此分钟和秒默认提供 0-59 全量选择；非法步长会回退到 1。
disabled / readonly / required / clearable 禁用、只读、必填和清空语义。只读可聚焦和查看弹层，但不可修改值。
size / variant / status / helper_text 尺寸、视觉变体、语义状态和辅助文本。
on_change / on_open_change / on_parse_error 用户交互回调。受控 set_* 同步不会触发这些回调。

公开方法

value() / mode() / is_open() / input_text() 读取当前值、模式、打开状态和输入文本。
set_value(...) / set_mode(...) 从外部同步值和模式，不触发用户变化回调。
set_disabled(...) / set_readonly(...) / set_status(...) / set_helper_text(...) 同步交互和展示输入。
set_min(...) / set_max(...) 同步可提交范围约束。
open(cx) / close(cx) / toggle(cx) / clear(cx) 执行用户语义操作。close 会丢弃未确认草稿。

键盘行为

  • Enter 在输入框中提交手动输入；日期面板打开时选择活动日期，纯时间模式打开时确认草稿。
  • Escape 关闭弹层并恢复到当前已提交值。
  • Left / Right 移动活动日期到前一天或后一天。
  • Up / Down 移动活动日期到上一周或下一周。
  • Home / End 移动活动日期到当前面板月份首日或末日。
  • PageUp / PageDown 切换上一月或下一月。
  • Shift+PageUp / Shift+PageDown 切换上一年或下一年。

第一版不包含时区、农历、周选择、季度选择、多日期选择、预设范围列表、远程禁用规则或自定义面板 renderer。
