Tree 标准树组件

Tree 支持嵌套节点、展开折叠、单选/多选、级联复选半选、过滤、 键盘导航、虚拟列表、禁用、状态样式、helper text、受控同步和明暗皮肤。

Nested Nodes Cascade Check Virtual List Keyboard Light / Dark

基础用法

Tree 是 gpui entity 组件，通常在父视图构造函数中通过 cx.new 创建，再在 Render 中作为子元素渲染。 

struct PermissionView {
    tree: Entity<Tree>,
}

impl PermissionView {
    fn new(cx: &mut Context<Self>) -> Self {
        let tree = cx.new(|cx| {
            Tree::new(
                cx,
                TreeProps::default()
                    .nodes(vec![TreeNode::new("admin", "后台管理").children(vec![
                        TreeNode::new("user.read", "查看用户"),
                        TreeNode::new("user.write", "编辑用户"),
                    ])])
                    .expanded_keys(vec![SharedString::from("admin")])
                    .checkable(true)
                    .helper_text(Some(SharedString::from("父子节点会自动计算 checked 和半选状态"))),
            )
        });

        Self { tree }
    }
}

TreeProps

属性 类型 说明
nodes Vec<TreeNode> 嵌套节点树。节点 key 要求唯一；重复 key 会保留第一次出现的节点。
expanded_keys Vec<SharedString> 初始展开节点 key。只有有子节点的 key 会影响可见行。
selected_keys Vec<SharedString> 初始行选中 key，语义由 selection_mode 决定。
checked_keys Vec<SharedString> 初始 checked key，会按级联规则规范化并派生半选节点。
filter_text SharedString 过滤文本，按 label 大小写不敏感匹配，并显示匹配节点的祖先路径。
selection_mode TreeSelectionMode None、Single、Multiple。
checkable bool 是否显示复选框并启用 checked 状态。
disabled / required bool 禁用状态和必填语义标记。
size / variant / status TreeSize / TreeVariant / TreeStatus 控制行高、容器样式和语义状态色。
helper_text / empty_text Option<SharedString> / SharedString 辅助文本和空状态文本。
max_height Pixels 虚拟列表最大视口高度，超过后内部滚动。

公开方法

Tree::new(cx, props) 创建 Tree 组件。
expanded_keys() / selected_keys() / checked_keys() 读取当前展开、选中和 checked key。
half_checked_keys() 读取当前半选 key，按当前节点树 DFS 顺序返回。
set_nodes(...) 从外部同步节点树，保留已有状态 key，不触发交互回调。
set_expanded_keys(...) / set_selected_keys(...) / set_checked_keys(...) 受控同步对应状态，不触发外部回调。
set_filter_text(...) / set_disabled(...) / set_status(...) / set_helper_text(...) 同步过滤、禁用、语义状态和辅助文本。
clear_selection(cx) / clear_checked(cx) 清空 selected 或 checked 状态，并按交互语义触发对应回调。
focus_handle() 返回焦点句柄，便于父组件主动聚焦。

键盘行为

  • Up / Down 移动活动行。
  • Home / End 移动到首个或最后一个可见节点。
  • Left 折叠当前节点；已折叠或叶子节点时移动到父节点。
  • Right 展开当前节点；已展开时移动到第一个可见子节点。
  • Enter 执行行选中。
  • Space 优先切换复选框；未开启复选时执行行选中。
  • Cmd+A / Ctrl+A 在多选模式下选中所有可见可选节点。

第一版不包含拖拽排序、异步加载、远程搜索、自定义节点渲染、右键菜单或节点编辑。
